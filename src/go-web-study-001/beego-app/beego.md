## beego

### Brief
- beego 是一个快速开发 Go 应用的 HTTP 框架，可以用来快速开发 API、Web 后端服务等各种应用，是一个 RESTful 的框架，主要设计灵感来源于 tornado、sinatra 和 flask 这三个框架，并结合了 Go 本身的一些特性（interface、struct 嵌入等）而设计的一个框架。

### feature

- MVC
- REST
- 智能路由
- 日志调试
- 配置管理
- 模板自动渲染
- layout设计
- 中间件插入逻辑
- 方便的JSON/XML服务
	

### Install

```shell
go get github.com/astaxie/beego

```
